1 Overview

This R Markdown contains the CHD1 CUT&RUN analysis steps relevant to the manuscript figures and statements:

  • Bowtie2 alignment-rate summary
  • Peak-calling workflow documentation
  • DiffBind setup and consensus CHD1 peakset generation
  • Peak annotation
  • Overlap between CHD1-bound genes and the top RNA-seq p-value decile
  • Export of gene lists used for downstream interpretation

Exploratory analyses and unused sections have been removed.

3 Alignment QC

The original FASTQ processing, Bowtie2 alignment, SAM/BAM conversion, bigWig generation, and MACS3 peak calling were performed outside R. The commands are retained here for reproducibility but are not evaluated during knitting.

# Run FastQC
find . -maxdepth 1 -mindepth 1 -type d -exec sh -c "cd '{}' && pwd && fastqc *.fastq.gz -t 90" \;

# Align paired-end reads to mm10 using Bowtie2
find . -maxdepth 1 -mindepth 1 -type d -exec sh -c "cd '{}' && \
  bowtie2 -p 40 --time --met-stderr --met-file bt2_metrics.txt \
  -x /hpcdata/Mimir/kimberl/mm10_bt2/mm10 \
  -1 '{$1}'_L008_R1_001.fastq.gz \
  -2 '{$1}'_L008_R2_001.fastq.gz \
  -S '{$1}'.sam" \;
# Convert SAM files to coordinate-sorted BAM files
for i in `ls */*.sam | rev | cut -c 5- | rev | uniq`
do
  samtools view -@ 16 -bS ${i}.sam | \
  samtools sort -T ${i} -O BAM -@ 16 -o ${i}.bam
done
# Merge IgG files for shared control
samtools merge -o all_merged_IgGs.bam -@ 48 *IgG*.bam

# Activate MACS3 environment
ml load Anaconda3/2022.05
conda activate /hpcapps/env/conda/MACS3

# Call CHD1 peaks against merged IgG control
for i in `ls *CHD1*.bam | rev | cut -c 5- | rev | uniq`
do
  echo "Calling peaks for sample: $i"
  macs3 callpeak \
    -t ${i}.bam \
    -c all_merged_IgGs.bam \
    -n ${i} \
    -f BAMPE \
    -g mm \
    --outdir ${i} \
    -q 0.1 \
    --max-gap 300 \
    --keep-dup all
done

# Generate RPGC-normalized bigWig files
for i in `ls *merged.bam | rev | cut -c 5- | rev | uniq`
do
  bamCoverage \
    --bam ${i}.bam \
    -o ${i}_depthnorm.bw \
    -p max \
    --binSize 5 \
    --smoothLength 30 \
    --normalizeUsing RPGC \
    --effectiveGenomeSize 2652783500 \
    --ignoreForNormalization chrX \
    --minFragmentLength 130 \
    --extendReads \
    --centerReads

6 DiffBind analysis

This section creates a DiffBind object, counts reads in peaks, applies blacklist/greylist filtering, normalizes, tests for differential CHD1 occupancy, and exports the WT DMSO consensus peakset.

final.samplesheet <- read.csv2("diffBind_sampleSheet_1.csv") %>%
  filter(str_detect(SampleID, "CHD1"))

dba.all <- dba(sampleSheet = final.samplesheet)
plot(dba.all)

dba.all <- dba.count(dba.all)
# Remove blacklist regions and generate/apply greylist
dba.all <- dba.blacklist(dba.all, greylist = FALSE)

# Normalize using native DiffBind methods with background signal
chd1.norm.background <- dba.normalize(
  dba.all,
  method = DBA_ALL_METHODS,
  normalize = DBA_NORM_NATIVE,
  background = TRUE
)

# Test for differential occupancy by condition
chd1.dba <- dba.analyze(
  chd1.norm.background,
  method = DBA_ALL_METHODS,
  design = "~Condition"
)
## coverage: 2872810 bp (0.11%)
dba.show(chd1.dba, bContrasts = TRUE)
